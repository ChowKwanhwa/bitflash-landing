BitFlash 参与教程

BitFlash Participation Tutorial

本教程包含 BitPocket 钱包安装及 BitFlash 代币购买流程

This tutorial covers BitPocket wallet installation and BitFlash token purchase process

第一部分：安装 BitPocket 钱包 (移动端)

Part 1: Install BitPocket Wallet (Mobile)

1 下载并安装 App Download and Install App
下载地址：Download URL:
https://www.bitdance.network/bitpocket.html

苹果手机请下载 TestFlight 版本；安卓手机直接下载 APK 安装包。

For iPhone, please download TestFlight version; For Android, download APK directly.

2 创建新钱包 Create New Wallet

打开 App，选择“创建新钱包”。如果您已有助记词，也可以选择“导入钱包”。

Open App, select "Create New Wallet". If you already have a seed phrase, you can select "Import Wallet".

3 设置安全密码 Set Security Password

设置一个安全的密码，用于日常解锁钱包。

Set a secure password for daily wallet unlocking.

4 备份助记词 (重要) Backup Seed Phrase (Important)

请务必手抄助记词并妥善保管！不要截图，不要发送给任何人。

Please write down your seed phrase and keep it safe! Do not screenshot or send to anyone.

5 获取充值地址 Get Deposit Address

进入钱包主页，点击“收款”按钮，获取您的 Taproot 比特币地址（以 bc1p 开头）。

Go to wallet home, click "Receive" button to get your Taproot Bitcoin address (starts with bc1p).

第二部分：购买 Bit Flash 代币

Part 2: Buy Bit Flash Token

1 进入 BitBoom 平台 Enter BitBoom Platform

打开 BitPocket 钱包，点击底部的“发现”页面。在列表中点击 "BitBoom.fun PRO" 进入平台。

Open BitPocket wallet, click "Discovery" tab at the bottom. Click "BitBoom.fun PRO" in the list to enter the platform.

2 连接钱包 Connect Wallet

点击页面右上角的 "Connect Wallet" 按钮，选择 BitPocket 钱包进行连接。

Click "Connect Wallet" button at the top right, select BitPocket wallet to connect.

3 选择 Bit Flash 代币 Select Bit Flash Token

在列表中找到 Bit Flash 代币，或者直接在搜索框搜索。点击进入交易详情页。

Find Bit Flash token in the list, or search directly. Click to enter trading details.

4 购买代币 Buy Token

1. 输入您想要购买的金额（SATS单位）。
2. 设置滑点（建议 2% 或更高以确保成功）。
3. 点击 "Buy Flash" 按钮确认交易。

1. Enter the amount you want to buy (in SATS).
2. Set slippage (Recommend 2% or higher to ensure success).
3. Click "Buy Flash" button to confirm.
